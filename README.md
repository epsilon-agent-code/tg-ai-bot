#  Сирена — ИИ-автоответчик для Telegram

**Сирена** — это умный ИИ-помощник, который отвечает за тебя в Telegram, когда ты занят. Работает на базе Qwen моделей через Alibaba DashScope API.

##  Возможности

###  Userbot (отвечает от твоего имени)
- Автоматические ответы когда ты занят (`/away`)
- Представляется как "Сирена, ИИ-помощник"
- Не раскрывает имя хозяина
- Естественный диалог с 3 уровнями промптов:
  - **1-й уровень**: первое сообщение (представление)
  - **2-й уровень**: в течение 5 минут (поддержка диалога)
  - **3-й уровень**: после паузы (напоминание о статусе)
- ИИ-судья: сам определяет спам и неадекватность
- Защита от спама (мут на 10 мин / 24 часа / навсегда)
- Команды управления

###  Публичный бот (отвечает как отдельный ИИ)
- Работает как @sirena_personal_bot
- Каждый пользователь использует СВОЙ API ключ
- Те же функции защиты и диалога

## 🚀 Установка Userbot (отвечает от твоего имени)

### Шаг 1: Установи Python 3.11

Скачай с https://www.python.org/downloads/release/python-3119/

При установке **ОБЯЗАТЕЛЬНО** поставь галочку **"Add Python to PATH"**

### Шаг 2: Получи API ключ Alibaba DashScope

1. Зайди на https://modelstudio.console.aliyun.com/
2. Зарегистрируйся (можно через Google/почту/телефон)
3. В меню слева нажми **API Key**
4. Нажми **+ Create API Key**
5. Выбери **Default Workspace** → **OK**
6. **СКОПИРУЙ ключ** (начинается с `sk-ws-...`)
    Ключ показывается **ТОЛЬКО ОДИН РАЗ**! Сохрани его сразу!

**Бесплатный лимит:** ~1000 сообщений в месяц для новых пользователей

### Шаг 3: Получи Telegram API данные

1. Зайди на https://my.telegram.org/
2. Введи свой номер телефона
3. Введи код из Telegram
4. Нажми **API development tools**
5. Заполни форму:
   - **App title:** любое (например "Sirena")
   - **Short name:** любое (например "sirena")
   - **URL:** оставь пустым
   - **Platform:** Desktop
6. Нажми **Create application**
7. Скопируй **API_ID** (число) и **API_HASH** (длинная строка)

### Шаг 4: Скачай проект

```bash
git clone https://github.com/ТВОЙ_НИК/tg-ai-bot.git
cd tg-ai-bot
```

Или скачай ZIP и распакуй.

### Шаг 5: Установи зависимости

```bash
pip install -r requirements.txt
```

### Шаг 6: Сгенерируй STRING_SESSION

Создай файл `generate_session.py`:

```python
from pyrogram import Client
import os

api_id = int(input("Введи API_ID: "))
api_hash = input("Введи API_HASH: ")

app = Client("my_userbot", api_id=api_id, api_hash=api_hash)

async def main():
    async with app:
        session = await app.export_session_string()
        print("\n✅ Твой STRING_SESSION:")
        print(session)
        print("\nСкопируй эту строку и вставь в .env файл!")

if __name__ == "__main__":
    app.run(main())
```

Запусти:
```bash
python generate_session.py
```

- Введи свой номер телефона
- Введи код из Telegram
- Скопируй полученную строку

### Шаг 7: Настрой .env

Создай файл `.env` в папке проекта:

```env
API_ID=твой_api_id
API_HASH=твой_api_hash
STRING_SESSION=твоя_строка_сессии
DASHSCOPE_API_KEY=sk-ws-твой_ключ
```

### Шаг 8: Запусти

```bash
python bot.py
```

Должно появиться:
```
🚀 Сирена запущена с ИИ-судьёй!
```

##  Использование Userbot

```
/away на экскурсии
```
или
```

```

### Выключить:
```

```

```

```

##  Публичный бот (отдельный ИИ-помощник)

### Шаг 1: Создай бота через @BotFather

1. Открой Telegram, найди **@BotFather**
2. Напиши `/newbot`
3. Придумай имя (например "Sirena")
4. Придумай username (должен заканчиваться на `bot`, например `sirena_my_bot`)
5. Скопируй **ТОКЕН** (длинная строка)

### Шаг 2: Добавь токен в .env

```env
BOT_TOKEN=1234567890:AAH_твой_токен
```

### Шаг 3: Запусти публичный бот

```bash
python public_bot.py
```

### Шаг 4: Как использовать

1. Найди своего бота в Telegram (например @sirena_my_bot)
2. Напиши `/start`
3. Бот даст инструкцию как получить API ключ
4. Отправь боту свой API ключ от Alibaba
5. Общайся!

**Каждый пользователь использует СВОЙ API ключ — ты не платишь за других!**

##  Защита

### ИИ-судья автоматически определяет:
- **Спам** → мут на 10 минут
- **Повторяющийся спам** → мут на 24 часа
- **Агрессия/оскорбления** → мут на 24 часа
- **Угрозы/троллинг** → мут навсегда

### Правила Сирены:
- ✅ Никогда не называет имя хозяина
- ✅ Представляется только при первом обращении
- ✅ Не использует эмодзи
- ✅ Отвечает кратко и по делу
- ✅ Естественное завершение диалога

##  Важные предупреждения

1. **НЕ публикуй `.env` файл!** В нём твои ключи
2. **НЕ делись STRING_SESSION** — это как пароль от аккаунта
3. **API ключ Alibaba** — каждый использует свой
4. **Userbot работает на твоём аккаунте** — не давай никому доступ

##  Проблемы и решения

### "Missing credentials"
Проверь что в `.env` нет пробелов вокруг `=` и кавычек

### "Invalid token"
Проверь что `BOT_TOKEN` полный (с цифрами перед `:`)

### "API key error"
Создай новый ключ на https://modelstudio.console.aliyun.com/

### Бот не отвечает
Убедись что включил режим `/away`
